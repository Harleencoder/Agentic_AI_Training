# Day 01 – Introduction to Agentic AI & Python Fundamentals

**Date:** 25 June 2026

## Overview

The first day of the Agentic AI Training focused on building a strong foundation in programming concepts, Python fundamentals, software architecture, and the evolution of Artificial Intelligence. We also explored how different programming languages interact with the operating system and learned the importance of problem-solving and optimization in software development.

## Topics Covered

- Introduction to Auribises Technologies
- Python Fundamentals
- Programming as Problem Solving
- Time Complexity and Space Complexity
- Program Execution Flow
- C/C++, Java, and Python Execution Models
- MVC (Model–View–Controller) Architecture
- Introduction to Artificial Intelligence
- Evolution of AI → ML → DL → LLMs

## Learning Outcomes

By the end of this session, I was able to:

- Understand programming as a structured problem-solving process.
- Learn the basics of Python and its execution model.
- Differentiate between compiled and interpreted programming languages.
- Understand how C/C++, Java, and Python interact with the operating system.
- Learn the purpose and components of MVC Architecture.
- Gain an overview of Artificial Intelligence and its evolution to Large Language Models.


## Key Takeaways

- Programming is fundamentally about solving real-world problems using logical thinking.
- Efficient software requires optimizing both execution time and memory usage.
- Different programming languages follow different execution models.
- MVC Architecture separates data, business logic, and user interface for better software design.
- Large Language Models form the foundation of modern Generative AI applications.
